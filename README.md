# Kidur-in-Tana

This repository is about enacting Kidur's principles with Tana the best tool we have right now - even if it is not (F)OSS.

Since, the goal is not ideological purity, but practical alignment: living Kidur's values as faithfully as possible with today's tooling, while staying open to transitioning toward more open and sovereign alternatives as they mature or we build them.
